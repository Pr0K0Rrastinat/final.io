Sleep Tracking

A mobile app to improve students' sleep

Problem

Many students face sleep problems due to stress and poor habits. We are developing an app to improve students' sleep, focusing on creating healthy habits.

My Role

I worked on data analysis regarding sleep, prototype development, and interface testing.

Approach to the Problem

We started by analyzing the current issues, interviewing users, and creating a set of scenarios to understand students' key needs regarding sleep.

Brainstorming

We generated ideas to improve the app. One idea was to integrate with fitness trackers for more accurate sleep tracking.

Sketches

We created initial sketches of the app's interface.

Wireframes

We created interface wireframes based on the sketches.

Prototype

We created an interactive prototype to test functionality.

Project Video

Usability Testing

Usability testing showed that users rated the interface positively, but suggested improvements for additional features.

Final Design

This is the app that helps students track their sleep and improve their routine.
